API Endpoints

  • GET /users - List all users
    Example URL: http://localhost:3000/users
  • POST /users - Create a new user
    Example URL: http://localhost:3000/users
    Example JSON Body: 
    {
  "name": "John Doe",
  "email": "john@example.com"
}
  • GET /users/:id - Get a user by ID
    Example URL: http://localhost:3000/users/1
  • PUT /users/:id - Update a user by ID
    Example URL: http://localhost:3000/users/1
    Example JSON Body: 
    {
  "name": "Jane Doe",
  "email": "jane@example.com"
}
  • DELETE /users/:id - Delete a user by ID
    Example URL: http://localhost:3000/users/1
  • GET /posts - List all posts
    Example URL: http://localhost:3000/posts
  • POST /posts - Create a new post
    Example URL: http://localhost:3000/posts
    Example JSON Body: 
    {
  "title": "My First Post",
  "content": "This is the content of the post",
  "UserId": 1
}
  • GET /posts/:id - Get a post by ID
    Example URL: http://localhost:3000/posts/2
  • PUT /posts/:id - Update a post by ID
    Example URL: http://localhost:3000/posts/2
    Example JSON Body: 
    {
  "title": "Updated Post Title",
  "content": "Updated post content",
  "UserId": 1
}
  • DELETE /posts/:id - Delete a post by ID
    Example URL: http://localhost:3000/posts/2
  • GET /posts?limit=&offset= - Get posts with pagination
    Example URL: http://localhost:3000/posts?limit=2&offset=1
  • GET /posts/search?query= - Search posts by title or author
    Example URL: http://localhost:3000/posts/search?query=world
  • GET /posts/search?query= - Search posts by title or author
    Example URL: http://localhost:3000/posts/search?query=world&sortBy=title&order=ASC
